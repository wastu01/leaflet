# About us
```
National Center for High-performance Computing (NCHC), founded in 1991, 
is Taiwan's only national-level supercomputing center. 
The NCHC plays a leading role in cloud technology that possesses a large computing, storage, 
networking and platform integration, providing the domestic users with the Could Integration Service of High Performance Computing (HPC), 
high quality networking, high efficiency storage, big data analysis and scientific engineering simulation.
```
